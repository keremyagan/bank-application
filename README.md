# bank-application
It is a simple banking example.  

Features such as Withdrawal / Deposit, Remittance/EFT are also available.  

Main Menu: 1-Sign Up 2-Sign In 3-Exit  

Submenu: 

1-Deposit Money 

2-Withdraw Money 

3-Remittance/Eft  

4-Money, Additional Money Transactions 

5-Account Details 

6-Change Password 

7-Exit
